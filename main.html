Política de Privacidade — Tab Volume by Domain

Última atualização: 24 de novembro de 2025

Visão geral

Esta extensão ajusta o volume de mídia (áudio/vídeo) por domínio e salva a preferência do usuário localmente no navegador. O objetivo desta política é explicar que dados (se houver) são coletados, por que são usados e como você pode gerenciá-los.

Quais dados coletamos

  • Valor de volume por domínio: apenas um número decimal entre 0.0 e 1.0 associado a um domínio (ex.: youtube.com).
  • A extensão lê o hostname/URL da aba ativa apenas para identificar onde aplicar o volume salvo; ela não armazena listas de URLs completas nem o conteúdo das páginas.

Finalidade do uso

Os dados acima são usados exclusivamente para:

  • Aplicar automaticamente a preferência de volume quando você visita um domínio para o qual salvou um valor.
  • Permitir que o usuário ajuste o volume via o popup e que essa preferência seja persistida entre sessões.

Sincronização

Se você estiver logado no Chrome e tiver a sincronização ativada, os valores de volume podem ser sincronizados via chrome.storage.sync entre seus dispositivos. Essa sincronização é gerenciada pela infraestrutura do Google/Chrome; a extensão não implementa nem controla servidores de sincronização.

Compartilhamento de dados

Não compartilhamos, vendemos ou transmitimos os dados coletados para servidores externos de terceiros pela extensão. O único armazenamento remoto possível é a sincronização via infraestrutura do Chrome/Google, caso esteja ativada pelo usuário.

Retenção

Os valores de volume são retidos até que o usuário os remova (por exemplo, usando o botão Redefinir no popup) ou até que o usuário limpe os dados no Chrome (limpeza de dados do navegador).

Segurança

Os valores são armazenados localmente pelo Chrome e protegidos pelas políticas de segurança do navegador. A extensão não transmite nem faz backup desses dados para servidores próprios.

Permissões

A extensão solicita as permissões seguintes no momento da instalação:

  • storage: para salvar as preferências de volume por domínio.
  • tabs e host_permissions: para identificar a aba ativa e o domínio onde aplicar o volume.
  • scripting/content_scripts: para acessar e ajustar elementos <audio> e <video> nas páginas (localmente).

Contato e solicitação de remoção

Se desejar remover dados, pedir exclusão ou tiver dúvidas sobre privacidade, entre em contato: seu-email@exemplo.com. Substitua este e‑mail pelo seu contato real antes de publicar.

Observações finais

Esta é uma política curta e direta para descrever o funcionamento da extensão. Se adicionar funcionalidades que envolvam envio de dados a servidores remotos, atualizaremos esta política e informaremos os usuários.

Versão curta para o painel do Web Store:

A extensão salva localmente (e opcionalmente sincroniza via Chrome) apenas um valor de volume (0.0–1.0) por domínio; não coleta nem envia dados pessoais para servidores externos.
