Tab Volume by Domain — Política de Privacidade

Última atualização: 25 de novembro de 2025

Nome da extensão

Tab Volume by Domain

Que dados coletamos

Coletamos apenas um valor numérico (decimal entre 0.0 e 1.0) por domínio para representar o volume preferido. Não coletamos nomes, senhas, conteúdos de páginas, ou qualquer dado pessoal.

Como usamos esses dados

Usamos o valor armazenado para ajustar automaticamente o volume de elementos de mídia (<audio> e <video>) no domínio correspondente quando você visita o site. Os dados são salvos localmente usando a API chrome.storage e podem ser sincronizados via Chrome Sync se o usuário tiver essa opção ativada.
