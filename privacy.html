Nome da extensão

Tab Volume by Domain

Que dados coletamos

Coletamos apenas um valor numérico (decimal entre 0.0 e 1.0) por domínio para representar o volume preferido. Não coletamos nomes, senhas, conteúdos de páginas ou qualquer dado pessoal.

Como usamos esses dados

Usamos o valor armazenado para ajustar automaticamente o volume de elementos de mídia (<audio> e <video>) no domínio correspondente quando você visita o site. Os dados são salvos localmente usando a API chrome.storage e podem ser sincronizados via Chrome Sync se o usuário ativar essa opção.

English (en)

Last updated: November 25, 2025

Extension name

Tab Volume by Domain

What data we collect

We collect only a numeric value (decimal between 0.0 and 1.0) per domain to represent the preferred volume. We do not collect names, passwords, page contents or any personal data.

How we use this data

We use the stored value to automatically adjust the volume of media elements (<audio> and <video>) on the corresponding domain when you visit the site. Data is stored locally using the chrome.storage API and may be synced via Chrome Sync if enabled by the user.
