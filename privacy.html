Extension name

Tab Volume by Domain

What data we collect

We collect only a numeric value (decimal between 0.0 and 1.0) per domain to represent the preferred volume. We do not collect names, passwords, page contents or any personal data.

How we use this data

We use the stored value to automatically adjust the volume of media elements (<audio> and <video>) on the corresponding domain when you visit the site. Data is stored locally using the chrome.storage API and may be synced via Chrome Sync if enabled by the user.
